# 🖼️ Assets Directory

This folder contains graphical assets and media resources used inside the Pokédex application.

---

## 📂 Subdirectories

### 1. 🎨 `types/`
Contains SVG files representing the icons for all 18 Pokémon types (e.g. `fire.svg`, `water.svg`, `grass.svg`).
- These SVGs are sourced from the Duiker101 Pokémon Icon repository.
- Loaded inside UI components using `tintColor` configuration to adapt dynamically.

### 2. 📱 `images/`
Contains system-generated Expo template assets, including application launchers, adaptive icons, splash screens, and default react logos.

### 3. 📱 `appmockup.png`
The main project mockups/banner displaying the primary Home and Detail page configurations.
